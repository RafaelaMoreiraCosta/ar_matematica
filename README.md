# AR Matemática

Projeto desenvolvido na **2ª Residência em Realidade Aumentada (RA)** do **Instituto iRede**, com atividades desenvolvidas na empresa **Virtuex**.


## Sobre o projeto

O **AR Matemática** é um aplicativo educacional desenvolvido em **Realidade Aumentada (RA)** com o objetivo de apoiar o ensino de geometria espacial para estudantes do Ensino Fundamental. O aplicativo permite a visualização e a interação com sólidos geométricos em 3D, proporcionando uma experiência de aprendizagem mais dinâmica, intuitiva e interativa.


## Problema

O ensino de conceitos de geometria espacial pode ser desafiador para muitos estudantes devido à dificuldade de visualizar objetos tridimensionais utilizando apenas materiais impressos ou imagens bidimensionais. Dessa forma, torna-se necessário o uso de ferramentas que tornem o processo de ensino e aprendizagem mais acessível e envolvente.


## Objetivo

Desenvolver uma ferramenta educacional utilizando Realidade Aumentada que auxilie professores e estudantes na compreensão de sólidos geométricos por meio da interação com modelos tridimensionais.


## Público-alvo

- Estudantes do Ensino Fundamental(5° ao 9° ano);
- Professores de Matemática.


## Funcionalidades

- Reconhecimento de marcadores de Realidade Aumentada;
- Visualização de sólidos geométricos em 3D;
- Interação com os modelos tridimensionais;
- Apoio às atividades desenvolvidas em sala de aula.


## Tecnologias utilizadas

- Unity 6
- C#
- AR Foundation
- ARCore XR Plugin
- Android SDK
- Git
- GitHub


## Como executar

1. Clone este repositório:

```bash
git clone <url-do-repositorio>
```

2. Abra o projeto utilizando a Unity.

3. Certifique-se de que as dependências do **AR Foundation** e do **ARCore XR Plugin** estejam instaladas.

4. Configure a plataforma **Android**.

5. Compile e execute o aplicativo em um dispositivo compatível com **ARCore**.
